在本周的"新闻焦点"里有两则新闻和微软的Java策略有关。一则是微软公
开宣称.NET不会支援Java，现在学Java的人是浪费时间; 另外一则是微软
和HP交换条件，将使用HP的Chai技术(详见下面"新闻焦点")。有趣的是微
软在明处放话抵制Java，在暗地又支援精简版Java ── HP的Chai。Java
这个东西，在各大厂商的眼中实在是千变万化，充满各种爱恨情仇。我们
就来看看各种不同角度的Java。

在微软眼中，Java简直是眼中钉肉中刺。有了Java，代表唯我独尊的
Windows 平台不再是唯一的选择。用Java开发出来的东西可以在各种平台
上执行，原本想藉著数量庞大的应用程式一步步扩大 Windows版图的计划
就此泡汤，再加上Linux的出现，Windows大好江山眼看情势不妙。所以微
软一看到Java就有气，.NET将支援十几种程式语言，就是死也不支援Java
。还自立门户弄了一套"全新程式语言"叫做C#，骨子里根本是Java，嘴上
却说跟Java一点关系也没有。Java之於微软，好比是紧箍咒之於孙悟空，
一听到就头痛眼花浑身不对劲。

对IBM来说，Java 也不是顶愉快的东西。这种跨平台的好技术不是出自老
大哥IBM，而是来自小老弟 Sun，而且得到业界的支援，这让IBM实在脸上
无光，不过为了对抗独大的微软，IBM也投下庞大人力财力在 Java上面，
希望藉Java找到自己的第二春。不料Sun推出Java 2，把JDBC、Servlets
、EJB、JSP等Server端技术统包成所谓的J2EE (Java 2 Enterprise 
Edition)，荣耀集於Sun一身。IBM 不爽Sun这样整碗捧去，不过为了顾全
大局，还是继续在 Java阵营扮演重要角色。对IBM来说，Java好比是蒋介
石眼中的台湾，虽然不尽满意，但是也无路可去，只有暂时全心投入建设
复兴基地，希望有朝一日当成反攻跳板光复神州。

而对Java的创始者Sun来说，Java就是未来的一切。如果没有Java，Sun只
是一个建造Unix伺服主机的公司。有了Java，让 Sun成为业界龙头，三天
两头上报纸科技版头条。Sun的野心当然不止於此，在Java 2推出後，Sun
要藉Java扩展版图的计划已经人尽皆知。藉J2SE (Java 2 Standard 
Edition)稳住Java在电脑软体上的应用，藉J2ME (Java 2 Micro Edition)
跨足更广大的家电、智慧卡、电子装置等市场，再藉J2EE (Java 2 
Enterprise Edition )一统伺服主机运算环境，阵仗一字排开，实在军容
壮盛。未来能否继续开疆拓土，就看 Sun能不能以更好的技术，加上更开
放的环境，吸引更多各产业的厂商加入这行列。所以在当前，也许UNIX伺
服主机是Sun主要营收来源，不过放眼未来，Java 技术才是永续发展的
